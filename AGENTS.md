## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for `Brucennnnn/jone-backend` using the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses this repo's GitHub labels, with `question` mapped to needs-info and `ready-for-agent` mapped directly. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain-doc layout. See `docs/agents/domain.md`.
