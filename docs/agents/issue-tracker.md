# Issue Tracker

This repo tracks work in GitHub Issues for `Brucennnnn/jone-backend`.

Use the `gh` CLI for issue operations.

Common commands:

- List issues: `gh issue list --repo Brucennnnn/jone-backend`
- View an issue: `gh issue view <number> --repo Brucennnnn/jone-backend`
- Create an issue: `gh issue create --repo Brucennnnn/jone-backend`
- Comment on an issue: `gh issue comment <number> --repo Brucennnnn/jone-backend`
- Close an issue: `gh issue close <number> --repo Brucennnnn/jone-backend`

Every issue or comment generated during triage must start with:

> *This was generated by AI during triage.*
